# نظام الإدارة المتكامل - تطبيق Flutter للجوال

تطبيق Flutter للجوال يحول نظام الإدارة المتكامل PWA إلى تطبيق محمول مع الحفاظ على جميع الوظائف.

## المميزات

### 1. الصفحة الرئيسية (Dashboard)
- عرض إحصائيات سريعة (عدد الأصناف، تنبيهات الصلاحية، عدد الملاحظات، الطلبات المعلقة)
- أزرار التنقل للأقسام الأربعة
- أزرار الاستيراد والتصدير (قريباً)

### 2. إدارة الجرد (Inventory)
- البحث عن الأصناف بالباركود أو الاسم
- عرض تفاصيل الصنف (الكود، الاسم، الكمية الحالية، كمية النظام)
- إضافة أو خصم الكميات
- حذف الأصناف
- جدول يعرض جميع الأصناف مع الفوارق

### 3. تنبيهات الصلاحية (Expiry)
- البحث عن الأصناف
- إضافة دفعات جديدة (تاريخ التحميص، مدة الصلاحية، الكمية)
- تعديل وحذف الدفعات
- جدول تفصيلي يعرض حالة الصلاحية (منتهي، قريب الانتهاء، صالح)
- تنبيهات للمنتجات التي ستنتهي خلال 30 يوم

### 4. دليل القهوة (Notes)
- البحث عن الأصناف
- إضافة/تعديل اسم المحمصة والإيحاءات
- جدول يعرض جميع الملاحظات

### 5. إدارة الطلبات (Orders)
- تبويبات (المعرض / الكافيه)
- إضافة طلب جديد (رقم الطلب، الشركة، نوع الطلب، التاريخ، ملاحظات)
- تعديل وحذف الطلبات
- تغيير حالة الطلب (منتظر / مكتمل)
- عرض مدة الانتظار مع تنبيه للطلبات المتأخرة (أكثر من 7 أيام)

## التقنيات المستخدمة

- **Flutter**: إطار العمل الأساسي لتطوير التطبيق
- **Provider**: لإدارة الحالة
- **SharedPreferences**: لحفظ البيانات محلياً
- **Intl**: للتعامل مع التواريخ والتنسيق العربي
- **Flutter Barcode Scanner**: لمسح الباركود (اختياري)

## المتطلبات

- Flutter SDK (3.0.0 أو أحدث)
- Dart SDK (يأتي مع Flutter)
- Android Studio أو VS Code مع ملحقات Flutter
- محاكي Android أو جهاز فعلي للاختبار

## التثبيت والتشغيل

### 1. تثبيت Flutter

قم بتحميل وتثبيت Flutter SDK من الموقع الرسمي:
```bash
https://flutter.dev/docs/get-started/install
```

### 2. التحقق من تثبيت Flutter

```bash
flutter doctor
```

### 3. استنساخ المشروع

```bash
git clone <repository-url>
cd inventory-pwa/flutter_app
```

### 4. تثبيت التبعيات

```bash
flutter pub get
```

### 5. تشغيل التطبيق

للتشغيل على محاكي أو جهاز متصل:

```bash
flutter run
```

للتشغيل في وضع التصحيح:

```bash
flutter run --debug
```

للتشغيل في وضع الإصدار:

```bash
flutter run --release
```

## بناء التطبيق

### بناء APK للأندرويد

```bash
flutter build apk --release
```

الملف سيكون متاحاً في:
```
build/app/outputs/flutter-apk/app-release.apk
```

### بناء App Bundle للأندرويد

```bash
flutter build appbundle --release
```

### بناء IPA لـ iOS

```bash
flutter build ios --release
```

## التخزين المحلي

التطبيق يستخدم SharedPreferences لحفظ البيانات محلياً:

- `ims_inventory_data_v2`: قائمة الأصناف مع الكميات والدفعات والملاحظات
- `ims_orders_data_v2`: طلبات المعرض والكافيه

## ملاحظات هامة

1. التطبيق يعمل بدون إنترنت (Offline-first)
2. البيانات محفوظة محلياً على الجهاز
3. الخطوط العربية مدعومة بالكامل
4. واجهة RTL (من اليمين لليسار)
5. ميزات الاستيراد والتصدير قيد التطوير
